"Object from Containers"

Detailed Description

Create container Object from a supported container.

  • The following constructors create an Object from one of the supported container types.
  • For the overloads with an l-value argument, the contained Objects are always copied.
  • For the overloads with an r-value argument, the contained Objects are copied iff the Object has a parent.
