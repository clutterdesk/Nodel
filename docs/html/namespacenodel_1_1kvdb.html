◆ register_directory_extension()

void nodel::kvdb::register_directory_extension ( const Object & fs_obj,
const std::string & ext = ".kvdb" )

Register a directory extension to recognize RocksDB database directories.

This function can be used to customize the extension used in a single tree, when that extension is different from the default extension.

Parameters
fs_objA filesystem object.
extThe directory extension, including the leading period.
Note
The extension is registered for the entire tree.
