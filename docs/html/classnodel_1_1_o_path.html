nodel::OPath

#include <Object.h>

Detailed Description

A simple path consisting of a list of keys.

Path literals can be created using ""_path literal operator.

◆ create()

Object nodel::OPath::create ( const Object & origin,
const Object & last_value ) const

Create objects necessary to complete this path.

Parameters
originThe starting point.
last_valueThe value that will become the leaf of the path.
  • Intermediate containers that do not exist are created with a container type that depends on the data type of the Key. If Key data type is an integer, then an ObjectList is created. Otherwise, an OrderedMap is created.
  • If the last_value argument has a parent then the copy that was added to the last container will be returned.
Returns
Returns last_value or a copy of last_value.
