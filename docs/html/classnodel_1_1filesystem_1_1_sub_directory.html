Determine the type of data in external storage.

Parameters
targetThe target holding this DataSource.
  • Implementations should override this method when the type of data in storage is dynamic.
  • Implementations should use metadata where possible to make this operation inexpensive. This method may be called in situations where the end-user does not want to incur the overhead of loading the data.
